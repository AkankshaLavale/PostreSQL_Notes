# PostgreSQL Basics

## What is PostgreSQL?

PostgreSQL is a powerful open-source relational database management system (RDBMS).

It is used to store, manage and retrieve data.

## Features

- Open Source
- Secure
- Supports SQL
- High Performance
- Used in real-world applications

## Why Learn PostgreSQL?

- Used in Data Analytics
- Used in Backend Development
- Used in Companies
- Strong alternative of MySQL
